# AI-Based Hate Speech Detection Using NLTK and Machine Learning

## Overview
This project detects hate speech, offensive comments, and normal comments on social media platforms using Machine Learning and Natural Language Processing.

## Technologies
- Python
- Flask
- NLTK
- Scikit-Learn
- MySQL
- Pandas
- Matplotlib

## Algorithms
- Naive Bayes
- SVM
- Logistic Regression
- Random Forest

## Features
- User Registration
- User Login
- Admin Login
- Dataset Upload
- NLTK Preprocessing
- AI Model Training
- Prediction System
- Prediction History
- User Warning & Blocking
